# 鑫港湾HIS项目

> 基于 .NET 8 + Vue 3 的医疗信息管理系统——从代码审查到上线交付的全流程工程化。

## 项目概述

鑫港湾HIS项目是面向基层医疗机构开发的医院信息系统。技术栈：.NET 8 + Vue 3 + MySQL 8.0 + Redis + Docker。代码托管于 Gitee（https://gitee.com/linhai-zhu/xingangwan-his-system）。当前处于开发阶段，尚未具备生产部署条件。

## 当前状态

- 已完成代码审查：235个问题（严重28 / 高64 / 中87 / 低56）
- 已分阶段规划：🔴 当下要改49个 / 🟡 上线前改85个 / 🟢 上线后优化101个
- 最紧迫问题：数据库迁移脚本阻塞开发环境初始化、访问控制缺失、敏感信息明文存储
- EC工业化规范手册 v2.8.0 已发布，作为交付质量标准
