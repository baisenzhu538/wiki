# Y模型探索营2：五次质变与科学推演

> 来源：一堂Y模型探索营第二课口述（Truman）。[[yt-personal-y-model-practice]] | [[yt-model-y-organization]] | [[一堂]]

## Summary

Y模型探索营2是实操篇的延续，Truman 首次完整公开自己与Y模型相关的**五次质变**——从观察子敬做科学投资，到憋出Y模型框架，再到"龙场悟道"发现Y模型是预测工具而非解释工具，最后到高阶营形成完整工作流。核心突破在于意识到：Y模型不是一个"事后诸葛亮"的解释框架，而是可以"事前诸葛亮"的推演引擎。关键动词仍是"牵"——以Y模型为线头，把前半生积累的全部认知串出来。

## Claims

### Truman 五次质变

| # | 质变事件 | 核心突破 |
|---|---------|---------|
| **1** | 观察子敬做科学投资 | 第一次意识到"科学是人类认识问题、解决问题的无限进步的思维方式" |
| **2** | 憋出Y模型（攻坚会） | 找到一个模型能解释所有科学做事的底层工具、原则、价值观 |
| **3** | 龙场悟道（某天晚上） | 发现Y模型不是解释工具，而是预测和指导工具——核心动词"牵" |
| **4** | NPC营交付 | 被提问启发，现场总结出工作流——后来用于辅导内部同学 |
| **5** | 高阶营 | 正式形成工作流 + 12345口诀，2天还原十几年成就感 |

### 关键区分：解释 vs 预测

| 用法 | 方向 | 难度 |
|------|------|------|
| **解释性使用** | 回头看——用Y模型解释已经发生的事 | 简单（什么都能套进去） |
| **预测性使用** | 向前看——用Y模型推导未知问题的解法 | 极难（需要大量积累+推演能力） |

> "怎么考都能考得进去。但是问题来了，怎么用Y模型来预测未来和指导真正的实践。"

### 12345口诀

| 数字 | 内容 |
|------|------|
| **1** | 抓住一个永远不变的东西：客观因果规律 |
| **2** | 两种认知姿态：实事求是 + 解放思想 |
| **3** | 三层推演：现象 → 模型 → 本质 |
| **4** | 四步工作流：识别问题 → 调用框架 → 牵出认知 → 推演方案 |
| **5** | 五个验证标准：逻辑自洽、实证支持、无反例、可预测、可迁移 |

### "牵"的深层含义

- 人脑中存储了大量方法论、工具、原则——但它们是散落的
- 面对新问题时，没有哪一个方法"刚好匹配"
- Y模型的作用：提供一个统一的逻辑起点，沿着它"牵"出所有相关知识
- 类比：Y模型 = 操作系统，其他方法论 = 应用程序

## Constraints & Boundaries


- claim:boundary-01 [conf=0.80] **"五次质变"叙事过于个人化**：每次质变都有高度特定的触发条件（观察子敬、攻坚会、NPC现场被提问），对普通学员的可复制路径不清晰。

- claim:boundary-02 [conf=0.80] **12345 口诀听起来像记忆术而非思维工具**：虽然 Truman 强调记住12345才能用Y模型，但记住口诀和能实操之间有巨大鸿沟——类似于记住游泳口诀和会游泳的区别。
