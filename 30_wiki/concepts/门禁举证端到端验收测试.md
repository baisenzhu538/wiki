# 门禁举证端到端验收测试

## Summary

这是一个用于验证 Sprint 2 全链路（ingest → enrich → gate）的临时测试素材。

KDO 目前完成了两个 Sprint： - Sprint 1：建立了 ingest + enrich + lint 基础管线 - Sprint 2：增加了 gate 门禁系统 + enrich 举证自动记录 三条核心管道命令： 每个 enrich 完成后自动生成举证记录到 `60_feedback/enrich-evidence/ev_*.

## Reusable Knowledge

- The KDO pipeline follows an ingest → enrich → gate flow, each stage executed via dedicated commands (`kdo ingest`, `kdo enrich --llm`, `kdo gate enrich`).
- Sprint 2 introduced the gate system and automatic recording of enrich evidence, building on the core pipeline established in Sprint 1.
- The `kdo enrich --llm` command performs a three-step compilation process to populate knowledge cards.
- After each enrichment, evidence records are automatically generated and saved to `60_feedback/enrich-evidence/ev_*.md`.
- The `kdo gate enrich` command validates that exit conditions for the enrichment phase are satisfied before proceeding.
- The system was developed incrementally: Sprint 1 delivered the base ingest–enrich–lint pipeline, and Sprint 2 added gating with automated evidence capture.
