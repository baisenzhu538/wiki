# 审美工具箱：永远以"最佳实践"为追求

> 来源：一堂泛产品设计审美工具箱指南 + 4张审美卡片。[[yt-model-pan-product-36-strategies]] | [[yt-model-pan-product-demand-toolkit]] | [[yt-personal-pan-product-aesthetics]]

## Summary

审美工具箱是泛产品设计36计的第二组卡牌（4张），回答"做成什么样才算好"。4张卡牌构成一条从主观到客观的审美升级链：收集（黑盒感觉）→ 建池（黑盒判断）→ 建模（白盒要素）→ 想象（超越最佳）。核心洞见：「你执行力的上限，不会超过你对最佳实践理解的下限」——审美不是天赋，是可训练的比较能力。

## [Condense]

### 审美升级四步法

```
主观琢磨 ←──────────────────────────→ 客观参考
    │                                       │
    ▼                                       ▼
1.最佳实践收集 ──→ 2.最佳实践池子 ──→ 3.最佳实践建模 ──→ 4.美好作品想象
  (主动调研)        (黑盒感觉)         (白盒要素)         (勇敢幻想)
  "找足够多"        "能判断好坏"       "能说清为什么好"    "能超越最佳"
    │                                       │
    └── 基础 ────→ 进阶 ────→ 成熟 ────→ 天花板 ──┘
```

### 卡牌详解

#### 1. 最佳实践收集 —— "找足够多"

| 维度 | 操作 |
|------|------|
| **搜集目标** | 复盘过去成功经验，设定Flag，找相关人/专家推荐 |
| **搜集渠道** | 人工搜集（定期跟进/落实到人）、系统搜集（平台/第三方/自己开发） |
| **日常习惯** | 世界学习法——随手沉淀亮点，养成「看到好的就收」的本能 |
| **整理工具** | 印象笔记/飞书知识库，尽量充分、MECE整理、持续跟进 |

#### 2. 最佳实践池子 —— "能判断好坏"

| 维度 | 操作 |
|------|------|
| **筛选入围** | 从大量收集中筛选，建立入围标准 |
| **常见池子类型** | 品牌名/slogan、logo/界面设计、文章/音乐/演讲/课程 |
| **对比打磨** | 同类对比，培养敏感性，建立黑盒判断力 |
| **建池子库** | 分类整理，扔进去试试，持续迭代 |

> 低端重复 vs 集中搜集 → 差距在于有没有一个"池子"

#### 3. 最佳实践建模 —— "能说清为什么好"

| 维度 | 操作 |
|------|------|
| **提炼N要素** | 使用白盒思维——找到大量最佳实践，逐个拆解直到能清晰说出要素 |
| **雷达图** | 完整评价维度（常用版/复杂版），建立稳定评价体系 |
| **获得反馈** | 找相关人验证模型，对比打磨 |
| **萃取总结** | 争取不比他们差——设定目标水准 |

> 这是从"这个好"到"为什么好"的关键跳跃。找到N要素 = 找到可复现的审美公式。

#### 4. 美好作品想象 —— "超越最佳实践"

| 维度 | 操作 |
|------|------|
| **沿画面想象** | 用户使用产品时最美最爽的体感、美好画面细节 |
| **沿张力想象** | 用户会怎么写评价/感言？推荐给朋友时会说什么？ |
| **沿迁移想象** | 某产品的XX感迁移过来会怎样？跨领域借鉴 |

> 有了最佳实践参考 + 抽象审美模型 → 动手能力会有一次跳跃式进步 → 更容易协作、复现和迁移创新

### 审美进步曲线

```
能力  ↑                    ┌─ 美好作品想象
      │                 ┌──┘
      │            ┌───┘  最佳实践建模
      │       ┌───┘
      │  ┌───┘    最佳实践池子
      │ ─┘ 最佳实践收集
      │
      └──────────────────────────→ 时间

低端重复 ──→ 集中搜集 ──→ 建池对比 ──→ 白盒建模 ──→ 超越创新
```

### 落地三原则

1. **集中搜集**：不要散落各处，建一个集中池
2. **MECE整理**：持续调整拆分/合并同类项，争取不重不漏
3. **持续跟进**：流程化、定期跟进，不靠一时冲动

## [Critique]

1. **"建模"环节的方法论支撑最弱**：从黑盒感觉到白盒要素的跳跃，课程中只给了"找到N要素"的抽象指导，缺少具体操作步骤（如如何进行要素分解、如何验证要素的完备性）。这可能是审美能力中最依赖天赋和经验的环节。

2. **池子的分类标准未结构化**：常见池子类型列了品牌名/logo/界面/文章/音乐等，但没有给出一个MECE的分类框架——不同类型的产品设计（如课程设计 vs 空间设计）的最佳实践池子结构应该不同。

3. **"美好作品想象"与"最佳实践建模"存在张力**：建模追求"不比他们差"，想象追求"超越最佳"——两阶段之间如何过渡？是先建模再超越，还是边建边超越？路径不够清晰。
