# 一堂创业地图：高潜力创业者修炼全景图

> 来源：一堂《创业地图：高潜力创业者修炼全景图》。[[一堂]] | [[yt-model-progress-map]] | [[yt-system-course-catalog#创业必修]]

## Summary

创业地图是进步大地图中「创业修炼」的完整展开，以「概率」为核心追求。按创业生命周期分为三个阶段：预判阶段（高水准选择）、起盘阶段（低成本创业）、增长阶段（可持续发展），底层是「实事求是 解放思想」的 Y 模型支撑。一句话：创业是个概率游戏，科学方法论帮你把自己修炼成概率高手。

## [Condense]

### 三阶段结构

| 阶段 | 核心命题 | 关键动作 | 对应课程模块 |
|------|---------|---------|-------------|
| **预判阶段** | 高水准选择 | 预判项目方向、预判行业机会、预判长期集中度、拆解关键假设 | 五步法体系、机会预判、行业画布 |
| **起盘阶段** | 低成本创业 | 高水平拆假设、十倍速验证假设 | 产品内核、单元模型、情报调研、精益实验 |
| **增长阶段** | 可持续发展 | 增长飞轮、融资加速、构建壁垒、战略取舍 | 融资必修、科学销售、科学抖音、渠道探索 |

### 起盘阶段双轨并行

| 轨道 | 目标 | 方法 |
|------|------|------|
| **商业拆解** | 把项目拆明白 | 挖需求 → 找内核 → 算模型 → 挖常识 |
| **高效验证** | 低成本证伪 | 找情报 → 做测试 |

### 增长阶段核心模块

| 模块 | 核心命题 |
|------|---------|
| 战略取舍 | 选择不做什么比选择做什么更重要 |
| 打造文化 | 公司文化的基因塑造 |
| 学习型组织 | 让组织具备持续进化能力 |
| 增长飞轮 | 找到增长的底层逻辑和杠杆点 |
| 融资加速 | 融资认知+实操体系 |
| 构建壁垒 | 项目壁垒的五步法分析 |

### 科学专项工具箱

六项成熟 + 五项研发中：

| 成熟专项 | 研发中专项 |
|---------|-----------|
| 科学决策、业务提升、关键转化 | 科学选品*、科学投放* |
| 科学销售、科学抖音、科学起名 | 科学选址*、科学定价*、科学私域* |
| 科学送礼 | |

### 底层逻辑

底部是八字方针的循环：**实事求是 ↔ 解放思想** → 持续练内功 → Y 模型：科学理念和追求

### 核心追求

**这张图的核心修炼目标是：概率**——创业是个概率游戏，不靠运气，靠把每一步的成功概率用科学方法提到最高。

## [Critique]

1. **「预判」与「起盘」的边界在实操中模糊**：很多项目在起盘阶段发现预判错误需要回头重判，但地图是单向箭头——缺少迭代回环的显性表达。
2. **科学专项工具的优先级指引缺失**：10+ 项专项工具，创业者应该先学哪个？哪些是特定阶段才需要的？
3. **「概率」作为核心追求缺少量化维度**：如何衡量我的"概率"提升了？从 10% 到 30%？还是靠感觉？地图未给出评估方法。

### 知识体系定位

创业地图是进步大地图中「创业修炼」的完整展开。与个人地图（修炼自己）和管理地图（修炼团队）不同，创业地图聚焦于「在不确定中做高概率决策」——从预判机会到低成本验证再到规模化增长的全链路方法论。
