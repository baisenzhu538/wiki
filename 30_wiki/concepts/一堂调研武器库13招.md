# 一堂调研武器库13招

## Summary

一堂调研系列第三节核心课，主讲商业调研的实操手段与信息获取技巧。课程提出「调研能力成长三阶段」（60分意识→70分方法论→80分技巧），并围绕「三端·内部·外部」三层信息地图，系统梳理了13个核心调研策略、5个必备概念、几十个小案例。

核心主张：调研最怕「以为自己原创、盲目自信、闭门造车」；信息价值100万时，值得花10万去拿；所有答案都在业务现场。

## Reusable Knowledge

### 一、五个必备概念

| 概念 | 定义 | 应用场景 |
|------|------|---------|
| **尽职调查（DD）** | 投资前对标的公司做全面调查（经营、财务、负债、合规） | 投资、并购、合作 |
| **逆向工程** | 拿到成品→倒推设计逻辑→为我所用 | 拆解竞品产品/成本 |
| **社会工程学** | 通过人的关系/信任/身份伪装获取信息 | 数据拿不到时"四两拨千斤" |
| **另类数据** | 非传统渠道数据（手机信号、地理数据、舆情监控） | 提前预判业绩 |
| **爬虫** | 写程序自动抓取公开数据 | 监控销量/库存/价格 |

### 二、调研地图（三层结构）

```
第一层：三端（最内核，最难拿，最值钱）
  ├─ 产品端（中间）
  ├─ 用户端（左边）
  └─ 合作方端（右边）

第二层：内部信息（可信度较高）
  ├─ 内线情报 / 人脉
  ├─ 求职工具
  └─ 假装身份

第三层：外部信息（最易拿，需交叉验证）
  ├─ 公开信息（官网/PR/招聘/财报）
  ├─ 专家网络
  ├─ 第三方数据库
  ├─ 研究报告
  └─ 其他公开渠道
```

### 三、13个核心策略

#### 策略1：完整体验产品
- 软件：逐功能试用；硬件：买回家上手；线下：扮演客户走完整流程
- **神秘顾客**：请人假装顾客记录服务流程（麦当劳/肯德基标准做法）
- 核心原则：很多人调研半天，连对手产品都没完整用过一遍

#### 策略2：产品数据分析（逆向分析）
- **观察公开数字**：订单号/ID自增数/销量/库存 → 定时记录推算增长
- **爬虫抓数据**：SKU、销量、价格、折扣 → 推导收入、市占率
- **实体拆解**：买竞品→拆开→分析元件/成本/设计逻辑（硬件行业惯例）
- **线下蹲点**：数人流、看成交、观察痛点（投资机构常用笨方法）

#### 策略3：访谈用户
- 四个核心问题：产品价值、使用过程、优点、缺点
- **追问技巧**："你有没有考虑其他品牌？"→ 扒出完整竞争清单
- 焦点小组、问卷调研（交给专业机构）

#### 策略4：口碑舆情收集
| 渠道 | 搜什么 | 看什么 |
|------|-------|--------|
| 搜索引擎 | "XX怎么样" | 论坛/UGC，滤掉PR稿 |
| 电商评论区 | 竞品商品页 | 好评点、差评点、高频词 |
| 应用商店 | APP评价 | 用户真实吐槽 |
| 社区论坛 | 知乎/小红书/垂直论坛 | 推荐和抱怨 |
| 黑猫投诉 | 品牌名 | 服务下限、集中槽点 |

→ 汇总后可画**雷达图**（用户在乎什么/不在乎什么）

#### 策略5：合作方研究
- 直接聊：供应商、物流、加盟商、服务商
- 上下游倒推：研究A公司的物流商B，通过B出货量倒推A
- **关键验证**：物流数据不好造假（张磊验网红案例：寄样品→找快递小哥→问发货量）

#### 策略6：内线情报 / 人脉
- 前员工（通常愿说真话）、现员工（脉脉/LinkedIn长线经营）
- **跟着投资机构去**：创始人面对投资人知无不言，跟着学
- 创始人/高管访谈：最高效但普通人难触达

#### 策略7：求职工具（大杀器）
| 方式 | 操作 | 获取什么 |
|------|------|---------|
| **高薪招聘** | 发高薪JD吸引对方核心人员面试 | 薪资结构、团队分工、SOP |
| **自己去面试** | 应聘对方关键岗位 | 技术路线、业务卡点 |
| **猎头中介** | 让猎头对接对方高管 | 高效触达+信息交换 |
| **入职学习** | 短期入职（尤其线下门店） | 单店模型、运营细节 |

> 💡 **技巧**：JD标高薪（能给20万标50万），来的人段位越高，诚意越足

#### 策略8：假装身份（社会工程学）
- 假装加盟商：聊成本结构、单元模型、选址规则
- 假装大客户：听销售话术、卖点、价格体系
- 假装家长/用户：套产品结构和定价
- 假装产品经理：站在竞品旁边访谈真实用户

> 口诀：**人生如戏，全靠演技，尊法守纪，学会戒急**

#### 策略9：公开信息
| 信息源 | 看什么 | 价值 |
|--------|-------|------|
| 官网 | 产品、定价、白皮书 | 基础认知 |
| 融资新闻 | 轮次、金额、里程碑 | 判断发展阶段（注意水分） |
| 招聘JD | 岗位方向、数量变化 | 推测战略转型、技术路线 |
| 一号位演讲 | 创始人公开课、采访 | 业务本质认知 |
| 财报/招股书 | 成本结构、增长驱动、壁垒 | 质量最高的公开教材 |

#### 策略10：专家网络
- 投资人/FA：每次融资都当成学习机会
- 行业老兵/创业者：找对赛道里泡了多年的人
- 付费平台：凯盛等专家网络（10万级预算）
- 在行/社交平台：花钱约聊

#### 策略11：第三方数据库
- **工商信息**：天眼查/企查查 → 股权结构、关联公司、融资历史
- **互联网档案馆**：看竞品官网历年版本变化
- **行业数据平台**：QuestMobile/蝉妈妈/新榜/各平台生意参谋

#### 策略12：研究报告
- 来源：券商研报、咨询公司报告、行业媒体、学术论文
- **注意立场**：券商为投资决策、媒体为品牌、有的拿钱办事
- **搜法**：关键词 + "行业报告"/"趋势报告"/"投资研究"

#### 策略13：其他公开渠道
- 新闻门户、行业会议、搜索引擎、垂直论坛
- **注意**：信息经多手传递易失真，**必须交叉验证**

### [Critique]

#### 前提假设评估
1. **"所有答案都在业务现场"**：假设物理/线下业务占主导。对纯线上SaaS、AI工具等轻资产业务，"蹲点"价值递减，产品体验和数据分析权重上升。
2. **"信息价值100万值得花10万拿"**：假设调研目标已精准锁定。若目标模糊，高投入可能导致沉没成本。
3. **"假装身份合法"**：课程多次强调"尊法守纪"，但社会工程学游走灰色地带，不同行业合规要求差异大（如医疗、金融）。

#### 边界与反例
- **反例**：某同学通过面试获取对方技术路线后，因双方技术栈差异过大，信息无法直接迁移，产生"有情报无行动"困境。
- **边界**：爬虫对抗反爬需要持续技术投入；另类数据（如手机信号）在国内合规性存疑。
- **边界**：上市公司财报质量高，但早期创业公司无财报可考，策略9对天使轮项目几乎失效。

#### 可靠性评估
- **高**：调研地图三层结构（三端→内部→外部）逻辑自洽，覆盖全面。
- **高**：13个策略均有真实学员案例支撑（银行设备蹲点、高薪招聘套SOP、假装加盟商等）。
- **中**：部分案例存在幸存者偏差——成功案例被强调，失败/踩坑案例提及较少。
- **低**：具体案例中的公司名称、数据均已脱敏，无法独立验证。
