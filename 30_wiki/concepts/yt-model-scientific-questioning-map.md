# 科学提问刻意练习地图

> 来源：一堂《科学提问·刻意练习成长地图》（内部使用）。[[一堂]] | [[yt-personal-deliberate-practice]] | [[yt-personal-deep-review]]

## Summary

一堂将提问能力按四个领域（业务分析→辅导团队→咨询教练→萃取经验）和三级难度分层，提供了每层的目标、场景、套路和工具箱。核心主张：提问不是天赋，是可以通过结构化模型刻意练习获得的核心能力。

## Claims

### 四领域 × 三级难度矩阵

| 领域 | 面向 | Level 1 | Level 2 | Level 3 |
|------|------|---------|---------|---------|
| **业务分析** (Analysis) | 面向业务 | 理解问题 → 共同评估 → 引导建模 | 对齐目标 → 强迫思考 → TopN 优先级 | 方案讨论 → 分析问题 → 交叉验证 |
| **辅导团队** (Coaching) | 面向团队 | 下属沟通 → 达成共识 → 代入场景 | 能力辅导 → 建设性提问 → 绩效沟通 | 面试评估 → 辩证讨论 |
| **咨询教练** (Teaching) | 面向客户 | 付费咨询 → 内核边界 → 关键假设 | 创业教练 → 苏格拉底式提问 | 私董会 → 探讨解法 |
| **萃取经验** (Socrates) | 面向专家 | 帮助客户提升找答案能力 | 公司萃取知识 | 个人做课 |

### 通用工具箱

- **业务分析**：STAR 模型、加减法、MECE 拆解
- **辅导团队**：GROW 模型、追问榨干经验、六个 Why
- **咨询教练**：关键假设、代入问题场景、苏格拉底式提问

### 提问的核心维度

| 维度 | 说明 |
|------|------|
| 提问目标 | 找出问题答案 / 提升员工找答案能力 |
| 提问对象 | 面向业务 / 面向团队 / 面向客户 / 面向专家 |
| 提问难度 | 青铜→白银→黄金→钻石→王者 |
| 使用场景 | 能力辅导、面试评估、绩效沟通、付费咨询、创业教练、私董会等 |

## Constraints & Boundaries


- claim:boundary-01 [conf=0.80] **四领域之间的交叉训练路径缺失**：业务分析能力与辅导能力如何在同一个练习循环中交替提升，未说明。
- claim:boundary-02 [conf=0.80] **难度分级的操作性不足**：青铜/白银/黄金/钻石/王者的区分标准过于模糊，需要可量化的晋级条件。
- claim:boundary-03 [conf=0.80] **工具箱的领域特异性**：STAR 和 GROW 跨领域使用是否有效，还是它们在各自领域内有不可替代的适配性。
