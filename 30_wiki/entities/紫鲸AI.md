# 紫鲸AI

> 面向中型品牌的AI内容营销操作系统——6个专精Agent编排为"数字员工团队"。

## 背景

紫鲸AI是面向中型品牌、MCN机构与区域连锁企业的AI内容营销平台。其核心创新在于将6个专精Agent编排为一支可开箱即用的数字员工团队，覆盖从品牌战略到私域运营的全链路自动化。

## 6 Agent管线

| Agent | 职能 |
|-------|------|
| AG1 定位大师 | 品牌定位与战略 |
| AG2 选题大师 | 内容选题策划 |
| AG3 文案大师 | 内容生成与文案 |
| AG4 运营总监 | 全域运营管理 |
| AG5 朋友圈运营总监 | 私域朋友圈运营 |
| AG6 社群运营总监 | 社群运营管理 |

## 技术架构

- **编排引擎**：LangGraph状态机，Checkpointer实现断点续传
- **Agent协议**：MCP/A2A双协议
- **品牌约束**：BrandKG品牌知识图谱（品牌词匹配率42%→89%）
- **定价**：基础订阅199-499元/月 + 超额按量 + 效果分成

## 与KDO的关联

紫鲸AI的"多Agent工作流编排"与KDO的深度调研集群方法论在多Agent协作模式上高度相关。其BrandKG知识图谱方案也为KDO的Graph RAG提供了"轻量级知识图谱落地"的参考路径。
