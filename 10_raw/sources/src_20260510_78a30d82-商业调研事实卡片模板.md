# 商业调研事实卡片模板

## 事实卡片格式

```markdown
## 事实 #[编号]

**陈述**：[具体事实描述，使用陈述句，不带主观判断]

**出处**：
- 来源：[平台/文档名称]
- 链接：[URL]
- 获取日期：[YYYY-MM-DD]

**信源层级**：L1 官方数据 / L2 权威报告 / L3 媒体报道 / L4 社区个人
**信息来源标注**：🔍实时搜索 / 🧠模型内置知识
**置信度**：✅ 高 / ⚠️ 中 / ❓ 低

**置信度判定依据**：
- ✅ 高：官方数据/财报明确披露，且有 ≥2 个独立来源确认
- ⚠️ 中：权威报告/媒体提及，但仅单一来源或未经官方确认
- ❓ 低：推测、匿名信息、或来自低可信度来源

**关联假设**：H[编号] — 支持 / 反对 / 中性
**备注**：[补充说明、矛盾信息、待验证点]
```

## 使用示例

```markdown
## 事实 #001

**陈述**：瑞幸咖啡 2024 年 Q3 营收为 101.81 亿元人民币，同比增长 41.4%。

**出处**：
- 来源：瑞幸咖啡 2024 年第三季度财报
- 链接：https://investor.luckincoffee.com/
- 获取日期：2024-11-20

**信源层级**：L1 官方数据
**信息来源标注**：🔍实时搜索
**置信度**：✅ 高

**置信度判定依据**：上市公司官方财报披露，SEC 公开文件

**关联假设**：H1（咖啡赛道仍在高速增长）— 支持
**备注**：需注意同店增速和新店贡献的拆分
```

## 事实卡片汇总表

| 编号 | 主题 | 信源层级 | 置信度 | 关联假设 | 验证方向 |
|------|------|---------|--------|---------|---------|
| #001 | 营收数据 | L1 | ✅ | H1 | 支持 |
| #002 | 门店数量 | L2 | ⚠️ | H2 | 中性 |

## 矛盾信息记录

```markdown
## 矛盾 #M001

**信息 A**：[来源 A 的说法]
**信息 B**：[来源 B 的说法]

**来源对比**：
- A：[信源层级 + 时间]
- B：[信源层级 + 时间]

**处理方式**：
- [ ] 采信更新/更权威的来源
- [ ] 两者保留，报告中说明分歧
- [ ] 需线下渠道进一步验证

**处理结果**：[最终采信哪方 + 理由]
```
