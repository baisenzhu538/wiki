## 交付检查清单

- [ ] **执行摘要可复述**：一句话结论清晰
- [ ] **假设全部验证**：无致命假设处于未验证状态
- [ ] **结论有证据**：每个核心结论有信源支撑
- [ ] **来源可追溯**：关键引用有链接
- [ ] **信源已分级**：L1-L4 标注完整
- [ ] **置信度标注**：整体和关键结论都已评估
- [ ] **风险已提示**：核心风险明确列出
- [ ] **盲区已声明**：未覆盖方面已说明
- [ ] **行动建议具体**：不是泛泛而谈
- [ ] **武器覆盖率标注**：用了多少个武器
- [ ] **Skill 联动出口已添加**
