# 商业调研报告 Style Guide（v1.0）

对标麦肯锡/BCG/Bain 视觉规范。Step 9 生成报告时严格遵守本文所有硬规范。

---

## 1. 配色体系（三色 + 中性）

| 角色 | HEX | 用途 |
|------|-----|------|
| **主色（McKinsey Blue）** | `#005EB8` | 一级标题、关键数据高亮、强调柱/线 |
| **辅色（蓝灰）** | `#4A90A4` | 二级标题、表头背景、次要数据 |
| **中性深灰** | `#595959` | 正文文字、坐标轴、非关键柱/线 |
| **中性浅灰** | `#BFBFBF` | 分隔线、坐标网格、图例 |
| **背景灰** | `#F5F5F5` | 引用块背景、侧栏 |
| **纯白** | `#FFFFFF` | 主背景 |

**颜色纪律**：
- 全文最多 4 色
- **灰色是主角**：全图表默认灰阶，仅强调一条数据用主色
- 警示色仅限风险段落：红 `#D32F2F`（致命风险）、黄 `#F9A825`（警告）、绿 `#2E7D32`（正面确认）

---

## 2. 字号三级体系

| 级别 | 字号 | 用法 |
|------|------|------|
| **H1 封面标题** | 28pt | 报告封面主标题（仅 1 个）|
| **H1 章节** | 24pt | 一级章节（"一、市场规模" 等）|
| **H2 小节** | 20pt | "1.1"、"1.2" 等 |
| **H3 子小节** | 16pt | "1.1.1" 等 |
| **正文** | 14pt | body 段落 |
| **表格/图注** | 12pt | 表格单元格、图表标签 |
| **页脚/来源** | 10pt | 引用、页码、来源说明 |

**字号纪律**：
- 任何情况**不得为挤字缩小字号**
- 正文行距 1.5 倍
- 标题上方空 2 行，下方空 1 行

---

## 3. 字体推荐

**中文**：
- 标题：思源宋体（Source Han Serif）或 鸿蒙黑体 Heavy
- 正文：思源黑体（Source Han Sans）Regular / 苹方 Regular

**英文**：
- 标题：Georgia / Merriweather（衬线）
- 正文：Arial / Helvetica Neue / Inter

**代码**：JetBrains Mono / Monaco

**规则**：全文最多 2 种字体。混排中英时，中英字体必须视觉高度一致（避免中文撑满英文空白）。

---

## 4. 数字格式（强制）

| 类型 | ✅ 正确 | ❌ 错误 |
|------|--------|--------|
| 金额（人民币）| `1,018 亿元` | `1018亿` / `1018 billion` |
| 金额（美元）| `$1.5B` 或 `15 亿美元` | `1.5 billion` / `1500M` |
| 百分比 | `23.5%` | `23%` / `23.47%` / `0.235` |
| 增速 | `↑ 15.3% YoY` | `+15%` / `增长 15.3` |
| 年份 | `2024` | `'24` / `2024年` / `FY24` |
| 季度 | `2024Q3` | `Q3 2024` / `3Q24` |
| 日期 | `2024-11-20` | `11/20/2024` / `20 Nov 2024` |
| 估值 | `估值 80 亿美元` | `valued at $8B` |

**规则**：
- 金额必须带千分位逗号
- 百分比保留 1 位小数（除非原始数据精度不足）
- 所有数字前后空格：`营收 1,018 亿元` 而非 `营收1,018亿元`

---

## 5. 表格规范

| 规范 | 要求 |
|------|------|
| **表头背景** | 辅色 `#4A90A4` 或浅灰 `#F5F5F5` |
| **表头字色** | 白色（深底）或黑色（浅底）|
| **交替行底色** | 浅灰 `#FAFAFA` 与白色 |
| **边框** | 仅上下两条主线（主色）+ 表头下划线；不要全包围 |
| **对齐** | 数字右对齐；文字左对齐；表头居中 |
| **单位** | 放表头括号内（例："营收（亿元）"），不放单元格 |
| **缺失数据** | 统一写 `—`（长破折号）而非 "N/A" / "无" / 空 |

---

## 6. 引用块（Executive Summary、关键洞察）

```markdown
> ### 📊 执行摘要
> 
> [SCQA 四段式内容]
> 
> | **KPI1** | **KPI2** | **KPI3** |
> |:-:|:-:|:-:|
> | 值 | 值 | 值 |
```

- 背景色 `#F5F5F5`
- 左侧加粗色条（主色 `#005EB8`）
- 内部字号与正文相同
- 图标：仅执行摘要用 📊，关键洞察用 💡，风险提示用 ⚠️

---

## 7. 分隔与节奏

- **章节间**用水平线 `---` 分隔（不要用 `===`）
- **段落间**一行空行
- **每章开头**有 1-2 句导语（不要直接进 H3）
- **每章结尾**有 1 段 3 行内的小结（用引用块或粗体前缀 "**小结**："）

---

## 8. 引用链接规范

**行内引用**：直接 MD 链接 `[来源名称](URL)`。

**脚注引用**（正式报告）：
```markdown
据瑞幸 2024 Q3 财报披露，营收同比增长 41.4%[^1]。

[^1]: 瑞幸咖啡 2024 年第三季度财报，https://investor.luckincoffee.com/，获取日期 2024-11-20
```

**参考资料章节**分 L1-L2（权威）与 L3-L5（次要）两组。

---

## 9. 图表硬约束（基于 Storytelling with Data + Tufte）

### 图表选型

| 数据类型 | ✅ 用 | ❌ 禁用 |
|---------|------|--------|
| 时间趋势 | 折线图 | 饼图、3D |
| 类别对比 | 条形图（从 0 开始，条宽 > 间距）| 双 Y 轴、3D |
| 部分对整体 | 100% 堆叠条（≤5 类）| 饼图（>5 类）、3D 饼 |
| 多维对比 | 热力矩阵 / 平行坐标 | 雷达图（>5 维）|
| 流程 | Sankey / 漏斗图 | 纯文字 |
| 关系 | 网络图 | 表格 |

### "灰色是主角" 原则

- 默认全图灰阶（`#595959`、`#BFBFBF`）
- 仅对需强调的那条数据用主色 `#005EB8`
- 不使用 Excel 默认的彩虹色

### 坐标轴规范

- Y 轴**必须从 0 开始**（除非是变化率）
- 时间轴间距一致
- 坐标轴标题必须带单位：`营收（亿元）`
- 删除网格线、边框、3D 效果
- 数据标签直接贴在柱/线末端（不用图例时）

### 字号

| 元素 | 字号 |
|------|------|
| Takeaway Title | 14pt 粗体 |
| 坐标轴标题 | 12pt |
| 坐标刻度 | 10pt |
| 数据标签 | 10pt |
| Source 来源 | 8pt 斜体 |

### 图表底部必备

每张图底部必须有：
```
Source: [来源名称，链接，获取日期] | N=[样本量] | 注：[特殊说明]
```

## 10. Takeaway Title 原则（BLOCKING）

**所有章节标题、图表标题必须是结论句**（含动词、含数字），不能是话题词。

| ❌ 话题式 | ✅ 结论式 |
|----------|----------|
| Q3 营收数据 | Q3 营收同比增长 41%，但同店增速仅 8% |
| 竞品对比 | 瑞幸在 6 个维度领先，但客单价落后星巴克 35% |
| 市场规模 | 中国现制咖啡市场 2024 年达 1,500 亿，CAGR 24% |
| 风险分析 | 核心风险为加盟商满意度仅 45%，低于行业均值 15pp |
| 团队背景 | 创始人有 3 次连续创业经验，其中 2 次成功退出 |

**Action Title 公式**：`[主体] + [动词] + [数字/方向] + [关键限定]`

**规则**：
- 标题 ≤ 20 字（中文）/ 15 词（英文）
- 必须包含至少 1 个数字或方向词（↑/↓/领先/落后/超过/低于）
- 读者仅看标题应能把握结论，无需读正文

---

## 11. 排版检查清单（Step 9.5 机械校验）

- [ ] 颜色数 ≤ 4
- [ ] 字体数 ≤ 2
- [ ] 所有 H2 标题为 Takeaway Title 格式（含数字或方向词）
- [ ] 所有数字符合第 4 节格式规范
- [ ] 所有表格有表头 + 来源 + 对齐正确
- [ ] 所有图表有 Takeaway Title + Source 脚注 + 坐标轴标题
- [ ] 无 Excel 默认彩虹色
- [ ] Y 轴从 0 开始（或明确标注截断）
- [ ] 缺失数据统一用 `—`

---
